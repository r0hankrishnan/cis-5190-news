03 - Stylistic Feature Engineering and Analysis¶

Research question: Can hand-engineered stylistic features classify Fox vs NBC headlines with competitive accuracy, without any topic information?

The EDA in notebook 02 established that Fox and NBC differ stylistically even when covering the same topics. This notebook quantifies those differences, selects the most discriminative features, and tests whether they can support classification on their own.

Sections:

  1. Candidate feature extraction (24 features across 5 categories)
  2. Feature comparison table (sorted by Fox vs NBC mean difference)
  3. Distribution plots for top features
  4. Feature selection rationale — which 7 make the cut
  5. Save style features for modeling

Result: 5 features selected. XGBoost on these features alone achieves % accuracy on the temporal test split — nearly on par with MiniLM+LR — using zero topic information.

1. Candidate Feature Extraction (24 features)¶

We compute a broad set of candidate style features spanning:

  • Punctuation/formatting — scare quotes, question marks, colons, ellipses
  • Capitalization — ALL CAPS words, title case ratio
  • VADER sentiment — compound score, positive/negative/neutral fractions
  • Attribution language — 'says', 'reports', 'allegedly', 'sources'
  • Structure — word count, starts with number, type-token ratio
  • spaCy NER — PERSON, ORG, GPE (place), NORP (political groups) counts and ratios

After computing all 24, we rank by absolute mean difference and select the ones with the largest and most interpretable gaps.

vader  = SentimentIntensityAnalyzer()
nlp_sp = spacy.load('en_core_web_sm')

def extract_all_features(headline: str) -> dict:
    text = str(headline)
    words = text.split()
    n_words = max(len(words), 1)
    vader_score = vader.polarity_scores(text)
    doc = nlp_sp(text)
    ent_types = [ent.label_ for ent in doc.ents]
    ent_counts = {t: ent_types.count(t) for t in ['PERSON', 'ORG', 'GPE', 'NORP']}
    n_ents = max(len(doc.ents), 1)

    return {
        # Punctuation / formatting
        'has_question':        int(text.endswith('?')),
        'scare_quote_count':   len(re.findall(r"'[^']{2,30}'", text)),
        'ellipsis_count':      text.count('...'),
        'exclamation_count':   text.count('!'),
        'has_colon':           int(':' in text),
        # Capitalization
        'allcaps_word_count':  sum(1 for w in words if w.isupper() and len(w) > 1),
        'title_case_ratio':    sum(1 for w in words if w.istitle()) / n_words,
        # VADER sentiment
        'vader_compound':      vader_score['compound'],
        'vader_neg':           vader_score['neg'],
        'vader_pos':           vader_score['pos'],
        'vader_neu':           vader_score['neu'],
        # Attribution / hedge language
        'has_says':            int(bool(re.search(r'\bsays?\b|\bsaid\b', text, re.I))),
        'has_report':          int(bool(re.search(r'\breports?\b|\breported\b', text, re.I))),
        'has_sources':         int(bool(re.search(r'\bsources?\b', text, re.I))),
        'has_allegedly':       int(bool(re.search(r'\ballegedly\b|\baccused\b|\bclaims?\b', text, re.I))),
        # Structure
        'n_words':             n_words,
        'starts_with_number':  int(bool(re.match(r'^\d', text))),
        'type_token_ratio':    len(set(words)) / n_words,
        # spaCy NER counts
        'person_count':        ent_counts['PERSON'],
        'org_count':           ent_counts['ORG'],
        'gpe_count':           ent_counts['GPE'],
        'norp_count':          ent_counts['NORP'],
        'total_ents':          len(doc.ents),
        # NER ratios
        'person_to_ent_ratio': ent_counts['PERSON'] / n_ents,
        'org_to_ent_ratio':    ent_counts['ORG'] / n_ents,
    }

print('Extracting 24 features...')
df_feat = df['title'].apply(extract_all_features).apply(pd.Series)

df_features = df_feat.copy()[[c for c in df_feat.columns if c not in ["datetime_posted", "title"]]]
feature_cols = [c for c in df_features.columns if c != 'is_fox']

summary = df_features.groupby('is_fox')[feature_cols].mean().T
summary = summary.rename(columns={0: 'NBC', 1: 'Fox'})

# Pooled std dev and Cohen's d -> from googling about how to compare two groups on multiple features
n_fox, n_nbc = len(df_features[df_features['is_fox'] == 1]), len(df_features[df_features['is_fox'] == 0])
pooled_std = np.sqrt(
    ((n_fox - 1) * df_features[df_features['is_fox'] == 1][feature_cols].std()**2 + (n_nbc - 1) * df_features[df_features['is_fox'] == 0][feature_cols].std()**2)
    / (n_fox + n_nbc - 2)
)

summary['abs_diff'] = (summary['Fox'] - summary['NBC']).abs()
summary['cohens_d'] = (summary['Fox'] - summary['NBC']) / pooled_std
summary['abs_cohens_d'] = summary['cohens_d'].abs()

summary = summary.sort_values('abs_cohens_d', ascending=False)

print('Feature means by source (sorted by |Cohen\'s d|):')

4. Feature Selection: 8 Key Features¶

We select features based on three criteria:

  1. Large absolute Cohen's D difference (top of the ranking above)
  2. Separable distributions (KDE or bar plots show clear separation, not heavy overlap)
  3. Interpretability (the feature has a plausible stylistic explanation)

Based on the criteria we chose the features with a Cohen's D above 0.35. We observe a sharp drop off in Cohen's D below 0.35 and many of the remaining features have inflated Cohen's D values due to it's low variance rather than an actual difference in mean values. The 5 features we chose also were clearly differentiated via their density/bar plots as well, further indicating that they present in a meaningfully different way in Fox vs NBC headlines.
